# LocationApp
O LocationApp é um projeto Android desenvolvido para explorar e demonstrar o uso dos componentes de localização nativos do Android. Este aplicativo permite entender e implementar funcionalidades de rastreamento e obtenção de dados de localização em tempo real.

### Principais Características e Aprendizados:
- Utilização de APIs de localização do Android para acessar dados de GPS.
- Utilização de permissões de sistema
- Implementação de rastreamento de localização em um aplicativo móvel.
- Integração de permissões necessárias para acesso à localização do dispositivo.

### Como Utilizar o Projeto:
1. Clone este repositório em sua máquina local.
2. Abra o projeto no Android Studio.
3. Execute o aplicativo em um dispositivo ou emulador Android.
4. Explore as funcionalidades de localização implementadas no aplicativo.
